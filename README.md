## 💡 使用说明

### 文件说明
- `.md` 文件：论文分析笔记模板，用于您的学习笔记
- `.txt` 文件：OCR提取的完整论文文本，便于搜索和阅读
- `.pdf` 文件：原始论文PDF，位于 `PDFs/` 文件夹

### 学习方法
1. 阅读 `.txt` 文件中的论文内容
2. 参考 `论文分析模板.md` 的结构整理笔记
3. 在对应的 `.md` 文件中填写笔记
4. 更新 `学习进度.md` 追踪学习进度

### 工具使用
- `scripts/` 文件夹中的Python脚本用于PDF处理和文件整理
- 如需重新OCR或整理文件，可运行相应脚本
