## 核心贡献

### 创新点1：统一的双系统VLA模型
- **问题描述**: 现有双系统方法将System 1和System 2保持为分离的模型，限制了System 1充分利用基于VLM的System 2的丰富预训练知识。
- **解决方案**: 提出Fast-in-Slow（FiS），一个统一的双系统视觉-语言-动作（VLA）模型，通过部分共享参数将System 1执行模块嵌入到基于VLM的System 2中。
- **意义**: 不仅实现System 1的高频执行，还促进推理和执行组件在System 2单一基础模型内的协调。

### 创新点2：异构模态输入和异步运行频率
- **问题描述**: System 1和System 2在FiS-VLA中具有根本不同的角色，需要不同的输入和运行频率。
- **解决方案**: 设计两个系统以整合异构模态输入和异步运行频率，实现快速和精确的操作。
- **意义**: 实现快速和精确的操作，同时保持System 2的上下文推理表示。

### 创新点3：双感知协同训练策略
- **解决方案**: 提出双感知协同训练策略，为System 1配备动作生成能力，同时保留System 2的上下文推理表示。
- **意义**: 实现两个系统之间的协调，同时保持各自的优势。

## 技术要点

### 架构设计
```
统一的双系统VLA模型:
  - System 2（基于VLM的推理模块）
    - 低频率运行
    - 上下文推理
    - 高级语义理解
    - 与System 1部分共享参数
  
  - System 1（执行模块）
    - 高频率运行
    - 快速动作生成
    - 嵌入在System 2中
    - 异步运行频率

异构模态输入:
  - System 2: 语言指令、视觉输入
  - System 1: 机器人状态、点云

双感知协同训练:
  - 为System 1配备动作生成能力
  - 保留System 2的上下文推理表示
  - 实现两个系统之间的协调
```

### System 1和System 2的协调
- **部分共享参数**: System 1嵌入在System 2中，部分共享参数
- **异步运行频率**: System 1高频率运行，System 2低频率运行
- **异构模态输入**: 两个系统使用不同的输入模态
- **协调机制**: 通过双感知协同训练实现协调

### 训练策略
- **双感知协同训练**: 为System 1配备动作生成能力，同时保留System 2的上下文推理表示
